Signal pipeline

Signal calculation flow

Live pages read stock_signal_latest. Historical validation uses stock_signal_snapshot and backtest result tables.

01 Collect source data

Quote, metric, SEC financials, candles, news, analyst data, and 13F flow are refreshed by batch jobs.

02 Validate data quality

Outliers, stale values, unit issues, missing coverage, and source conflicts are flagged before scoring.

03 Calculate factor scores

Each factor is normalized to 0-100 and stored with reasons, freshness, and data-quality context.

04 Build integrated Signal

Weighted scoring plus guardrails prevents weak quality, earnings, or momentum from receiving excessive scores.

05 Save snapshots

List and detail pages read persisted values, so failed batch runs do not erase the last valid Signal.

06 Validate outcomes

5D/20D/60D returns, factor diagnostics, walk-forward, and portfolio backtest test the usefulness of the score.
